# prejeto-academia-evolution
🏋️ Academia Evolution

Projeto desenvolvido durante meus estudos de HTML.

📖 Sobre o projeto

Este projeto simula o site de uma academia, com informações sobre exercícios, planos e um formulário de inscrição de alunos.

O objetivo principal foi praticar a estruturação de páginas web utilizando HTML, com foco em organização de conteúdo e uso de tags semânticas.

🛠️ Tecnologias utilizadas
HTML5
📚 O que foi praticado
Estrutura básica de páginas HTML
Uso de tags semânticas (header, main, section, footer)
Criação de menus de navegação com nav, ul, li e a
Organização de conteúdo em seções (section)
Uso de imagens com img
Criação de tabelas com table, tr, th e td
Criação de formulários com form, label, input, select, option, textarea e button
Uso de radio e checkbox
Estruturação de hierarquia de conteúdo
🎯 Objetivo de aprendizado

Este projeto faz parte da minha evolução como desenvolvedor front-end, com foco inicial em HTML. Ele serve como base para o aprendizado de CSS e JavaScript nos próximos passos.
